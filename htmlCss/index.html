Display dan Dimension

Tag Div dan Span

Tag div dan span digunakan untuk mengelompokan elemen html atau memberi style secara spesifik.

Default display mode

Tag div bersifat block.

Tag span bersifat inline.


Opsi nilai display

block
  • membuat baris baru saat dirender
  • dapat diset lebar dan tingginya
  • bila lebar tidak diset, maka lebar elemen akan memenuhi lebar dari parentnya
  • di dalamnya dapat disimpan elemen bertipe block, inline dan inline-block

inline
  • tidak membuat garis baru saat dirender
  • lebar dan tinggi sebesar konten yang ada di dalamnya
  • lebar dan tingginya tidak dapat diatur
  • margin dan padding hanya mempengaruhi elemen secara horizontal dan tidak vertikal.

inline-block
  • tidak ada elemen html yang default bertipe inline-block
  • nilai property display dapat diset manual menjadi inline-block
  • sifat dasar sama dengan inline
  • inline-block akan berpengaruh dari property width dan height.

none

digunakan untuk menyembunyikan elemen dari layar browser.

Box Model

Setiap elemen dihalaman html dipresentasikan sebagai sebuah kotak. Kita bisa mengatur ukuran maupun posisi kotak tersebut.

Komponen Box Model

Margin Jarak spasi antara satu elemen dengan elemen lain atau elemen parentnya. Margin bersifat transparan.

Border batang garis yang mengelilingi padding dan content.

Padding jarak spasi antara border dan content. Padding bersifat transparan.

Content isi dari elemen, bisa berupa text atau elemen lain. Text, gambar atau elemen lainnya ditampilkan dibagian content.


Property margin, padding dan border.

margin padding border
margin-top | margin-right | margin-bottom | margin-left | margin padding-top | padding-right | padding-bottom | padding-left | padding border-top | border-right | border-bottom | border-left | border
px | % | auto px | % | note: padding tidak boleh bernilai negatif cth: border: width style color;
width: ukuran ketebalan border (1px, 2px, 3px, ...)
style: bentuk garis (dotted, dashed, solid)
color: warna dari border

Box Sizing

content-box

Apabila menggunakan content-box, maka lebar box akan dijumlahkan dengan padding.

Misal lebar konten 160px dengan padding-left 40px dan padding-right 40px, maka total lebar box tersebut menjadi 240px.

border-box

Pada border-box ukuran sudah ditentukan 160px, tidak akan dijumlahkan dengan padding yang kita tentukan.


Overflow

auto

Pada overflow auto akan ditambahkan scroll apabila konten yang ada melebihi box yang disediakan, namun bila konten yang ada tidak melebihi box. Maka tidak akan ditambahkan scroll.

scroll

Pada overflow scroll, setiap box akan ditabahkan scroll atau tidak memandang apakah koten yang ada sedikit ataupun banyak.

hidden

Pada overflow hidden, apabila konten yang ada banyak dan melebihi box. Maka konten sisanya tidak akan ditampilkan.

CSS Float

css float digunakan untuk memposisikan elemen dalam sebuah layout halaman website. Property elemen float: left, right, none. Secara default float memiliki nilai none.

float: none; menentukan bahwa sebuah elemen tidak boleh mengapung(float).

float: left; digunakan untuk menentukan bahwa sebuah elemen harus mengapung(float) disebelah kiri dari elemen blok yang menampungnya.

float: right; digunakan untuk menentukan bahwa sebuah elemen harus mengapung(float) disebelah kanan dari elemen blok yang menampungnya.

CSS Position

CSS position terdiri dari 4 nilai, yaitu: static, relative, absolute, fixed.

default position: static.

static

position: static;

  1. static adalah niali default dari tipa-tiap elemen ketika tidak diberi property position.
  2. Menggunakan position selain static (non-static) akan membuat sebuah elemenmenjadi seolah olah berbeda dimensi dari elemen lainnya.
  3. Elemen yang diberi position selain static dapat menggunakan properti top, left, bottom, right untuk mengatur posisinya.

relative

position: relative;

  1. ketika kita menggerakkan elemen dengan posisi relative(top, left, bottom, right) ruang yang ditempati elemen tersebut masih ada.
  2. ketika kita menggerakkan elemen dengan posisi relative, elemen akan bergerak relatif terhadap posisi semulanya.
  3. jika kita memberi properti top: 0; dan left: 0; maka elemen tidak berubah posisinya.

absolute

position: absolute;

  1. ketika kita menggerakkan elemen dengan posisi absolute(top, left, bottom, right), ruang yang ditempati oleh elemen tersebut dianggap tidak ada.
  2. ketika kita menggerakkan elemen dengan posisi absolute, elemen akan bergerak relatif terhadap posisi dan elemen parentnya. selama elemen parentnya memiliki posisi yang juga non-static.
  3. jika kita memberi properti top:0; dan left:0;, maka elemen akan berada di ujung kiri atas dari elemennya.

fixed

position: fixed;

  1. ketika kita menggerakkan elemen dengan posisi fixed(top, left, bottom, right), ruang yang ditempati oleh elemen tersebut dianggap tidak ada.
  2. ketika kita menggerakkan elemen dengan posisi absolute, elemen akan bergerak relatif terhadap window(browser), meskipun elemen parentnya bernilai non-static.
  3. jika kita memberi properti top:0; dan left:0; maka elemen akan berada di ujung kiri atas dari elemen parentnya.
  4. elemen akan terkunci dan tidak bergerak dari posisinya meskipun halaman di scroll.

CSS Background

property background dalam css digunakan untuk mendefinisikanefek background dalam suatu elemen.

property background:

background-color

background-color digunakan untuk menenetukan warna background dari sebuah elemen.

Di dalam css, warna ditentukan oleh:

  • HEX value = seperti #ff0000
  • RGB value - seperti rgb(255, 0, 0)
  • nama warna - seperti red

background-image

background-image digunakan untuk menentukan bacground berupa gambar.

contoh penggunaannya: bacground-image: url(image.jpg);

background-repeat

background-repeat digunakan untuk mengulang gambar secara horizontal dan vertikal. gambar dapat diulang dengan menggunakan repeat-x atau repeat-y

contohnya: background-repeat: repeat-y;

background-position

background-position digunakan untuk menentukan posisi background. value yang dapat digunakan: left top, left center, left bottom, right top, right center, right center, right bottom, center top, center center, center bottom.

contohnya: background-position: center;

CSS Font

Font Family

font-family digunakan untuk mengatur jenis font yang akan digunakan.

pada css, font-family dikelompokkan menjadi 2 bagian:

  • gengeric family - adalah jenis font standar yag dipersiapkan sebagai pengganti jika font-family yang kita pilih tidak tersedia dalam sistem komputer kita.
  • family name / nama font - adalah kelompok font yang memiliki jenis serupa seperti Times New Roman atau Arial, dll.

NB: jika kita menetapkan family name, maka sebaiknya kita juga menambahkan generic family di akhir penulisan seperti font-family mengingat adanya kemungkinan jenis font yang tidak tersedia.

menentukan nilai relatif dari sebuah elemen terhadap elemen html.

CSS Animation

digunakan untuk menambahkan animasi pada web.

browser pendukung:

property animation
chrome 43.0
4.0 -webkit-
Internet Explore / Edge 10.0
firefox 16.0
5.0 -moz-
safari 9.0
4.0 -webkit-
opera 30.0
15.0 -webkit-
12.0 -o-

keterangan:

angka-angka dalam table menentukan versi browser pertama yang sepenuhnya mendukung property.

angka-angka yang diikuti oleh -webkit-, -moz-, -o- menentukan versi pertama yang bekerja dengan awalan -webkit-, -moz-, -o-.

contoh pengguna opera versi 30.0 ke atas:

animation-iteration-count: 2;

sedangakan pengguna opera versi 15.0 hingga 29.0:

-webkit-animation-iteration-count: 2;


nilai properti:

animation-name menentukan nama keyframe yang akan dikaitkan ke slector.

animation-duration menentukan berapa lama waktu(dalam satuan detik atau milidetik) yang diperlukan untuk menyelesaikan animasi.

animation-timing-funtion menentukan kurva kecepatan animasi.

  1. linear mengatur animasi dengan kecepatan yang sama dari awal hingga akhir.
  2. ease nilai default animation-timing-funtion yaitu animasi memiliki kecepatan awal lambat, kemudian cepat, kemudian berakhir perlahan.
  3. ease-in mengatur animasi dengan kecepatan awal atau permulaan yang lambat.
  4. ease-out mengatur animasi dengan akhir yang lambat.
  5. ease-in-out mengatur animasi dengan awal yang lambat dan akhir yang lambat.
  6. initial menetapkan properti ke nilai default.
  7. inherit mengikuti nilai properti dari elemen induk, misalkan pda elemen induk di set ease-in-out maka akan ikut di set ease-in-out juga.

animation-delay menentukan penundaan atau jeda sebelum animasi dimulai.

animation-iteration-count menentukan berapa kali animasi harus dimainkan.

animation-direction menentukan apakah animasi harus dimainkan ke depan, mundur atau gabungan maju dan mundur(alternate). nilai propretinya:

  1. normal animasi dimainkan seperti biasa (kedepan).
  2. reverse animasi dimainkan secara tebalik (mundur).
  3. alternate animasi dimainkan ke depan terlebih dahulu, lalu mundur.
  4. alternate-reverse animasi diputar mundur terlebiuh dahulu, lalu ke depan.
  5. initial mengatur animasi ke nilai default.
  6. inherit mengikuti nilai propeti dari elemen induk, misalkan pada elemen induk di set reverse maka akan ikut di set reverse juga.

animation-fill-mode menentukan gaya sebuah elemen ketika animasi tidak sedang dimainkan(sebelum mulai, setelah berakhir, atau keduanya).

animation-play-state menentukan apakah animasi berjalan atau berhenti.


More Pseudoclass

digunakan untuk menentukan style khusus suatu elemen:

:first-child

menentukan style khusus pertama dari pada banyaknya list.

contohnya: li:first-child{value..}

:last-child

kebalikan dari first-child, menentukan style khusus elemen yang terakhir.

contohnya: li:last-child{value..}

:nth-child(n)

menentukan style khusus elemen yang diinginkan, misal elemen dengan urutan 2 dan 3.

contohnya: li:nth-child(2), li:nth-child(3){value..}

:nth-child(odd) dan :nth-child(even)

menentukan style khusus elemen ganjil dan genap.

contohnya: li:nth-child(odd){background-color: yellow; color:red;} dan li:nth-child(even){background-color: orange;}

:first-letter

menentukan style khusus untuk huruf pertama pada suatu elemen.
