Display dan Dimension

Tag Div dan Span

Tag div dan span digunakan untuk mengelompokan elemen html atau memberi style secara spesifik.

Default display mode

Tag div bersifat block.

Tag span bersifat inline.


Opsi nilai display

block
  • membuat baris baru saat dirender
  • dapat diset lebar dan tingginya
  • bila lebar tidak diset, maka lebar elemen akan memenuhi lebar dari parentnya
  • di dalamnya dapat disimpan elemen bertipe block, inline dan inline-block

inline
  • tidak membuat garis baru saat dirender
  • lebar dan tinggi sebesar konten yang ada di dalamnya
  • lebar dan tingginya tidak dapat diatur
  • margin dan padding hanya mempengaruhi elemen secara horizontal dan tidak vertikal.

inline-block
  • tidak ada elemen html yang default bertipe inline-block
  • nilai property display dapat diset manual menjadi inline-block
  • sifat dasar sama dengan inline
  • inline-block akan berpengaruh dari property width dan height.

none

digunakan untuk menyembunyikan elemen dari layar browser.

Box Model

Setiap elemen dihalaman html dipresentasikan sebagai sebuah kotak. Kita bisa mengatur ukuran maupun posisi kotak tersebut.

Komponen Box Model

Margin Jarak spasi antara satu elemen dengan elemen lain atau elemen parentnya. Margin bersifat transparan.

Border batang garis yang mengelilingi padding dan content.

Padding jarak spasi antara border dan content. Padding bersifat transparan.

Content isi dari elemen, bisa berupa text atau elemen lain. Text, gambar atau elemen lainnya ditampilkan dibagian content.


Property margin, padding dan border.

margin padding border
margin-top | margin-right | margin-bottom | margin-left | margin padding-top | padding-right | padding-bottom | padding-left | padding border-top | border-right | border-bottom | border-left | border
px | % | auto px | % | note: padding tidak boleh bernilai negatif cth: border: width style color;
width: ukuran ketebalan border (1px, 2px, 3px, ...)
style: bentuk garis (dotted, dashed, solid)
color: warna dari border

Box Sizing

content-box

Apabila menggunakan content-box, maka lebar box akan dijumlahkan dengan padding.

Misal lebar konten 160px dengan padding-left 40px dan padding-right 40px, maka total lebar box tersebut menjadi 240px.

border-box

Pada border-box ukuran sudah ditentukan 160px, tidak akan dijumlahkan dengan padding yang kita tentukan.


Overflow

auto

Pada overflow auto akan ditambahkan scroll apabila konten yang ada melebihi box yang disediakan, namun bila konten yang ada tidak melebihi box. Maka tidak akan ditambahkan scroll.

scroll

Pada overflow scroll, setiap box akan ditabahkan scroll atau tidak memandang apakah koten yang ada sedikit ataupun banyak.

hidden

Pada overflow hidden, apabila konten yang ada banyak dan melebihi box. Maka konten sisanya tidak akan ditampilkan.

CSS Float

css float digunakan untuk memposisikan elemen dalam sebuah layout halaman website. Property elemen float: left, right, none. Secara default float memiliki nilai none.

float: none; menentukan bahwa sebuah elemen tidak boleh mengapung(float).

float: left; digunakan untuk menentukan bahwa sebuah elemen harus mengapung(float) disebelah kiri dari elemen blok yang menampungnya.

float: right; digunakan untuk menentukan bahwa sebuah elemen harus mengapung(float) disebelah kanan dari elemen blok yang menampungnya.
