Javascript

Pengertian Javascript

Javascript adalah bahasa pemrograman yang digunakan untuk menambah fitur interkatif pada website, seperti ketika ingin membuat game, melakukan perubahan ketika mengklik tombol, efek dinamik, animasi, dan masih banyak lagi.

Javascript diciptakan oleh Brendan Eich yang juga merupakan co-founder dari Mozilla project, Mozilla Foundation dan Mozilla Corporation.

Dukungan Browser

Javascript bisa dijalankan di browser Google Chrome, Internet Exploler atau Microsoft Edge, Mozilla Firefo, Opera, Safari.

javascript berbeda dengan Java.

Javascript biasanya di jalankan di console.

Variable dan Type Data Javascript

Variable adalah digunakan untuk menampung data atau konstanta di memori yang memiliki nilai.

var nama_variable = nilaidata;

var x = 5;

var y = 2;

var z = x+y;

Type data digunakan untuk menentukan jenis suatu data (deklarasi variable).

Jenis type data:

  • string = "hallo"
  • boolean = true / false
  • number = 1,2,3
  • array = ["hallo", "dunia", ...]
  • object = {hallo: "dunia", ...}

cara mengecek type data dapat menggunakan = typeof nama_variable

Opeartor

Operator adalah simbol-simbol yang digunakan untuk mengoperasikan suatu nilai data (operand).

macam operator:

operator aritmatika

digunakan untuk melakukan aritmatika pada angka.

  • + | penambahan
  • - | pengurangan
  • * | perkalian
  • / | pembagian
  • % | modulus / sisa bagi
  • ++ | increment
  • -- | decrement

operator assigment

memberikan nilai ke variable javascript.

  • = | x = y | x = y
  • += | x += y | x = x + y
  • -= | x -= y | x = x - y
  • *= | x *= y | x = x * y
  • /= | x /= y | x = x / y
  • %= | x %= y | x = x % y

operator penggabung

digunakan untuk mneggabungkan string.

var teks1 = "hallo";

var teks2 = "dunia";

var teks3 = teks1 + " " + teks2;

operator pembanding

  • == sama isinya
  • === sama isinya dan sama tipenya
  • != tidak sama dengan isinya
  • !== tidak sama isinya dan tidak sama tipenya
  • > lebih dari
  • < kurang dari
  • >= lebih dari sama dengan
  • <= kurang dari sama dengan
  • ? operator ternary (if else sederhana)

operator logika

  • && AND bernilai true
  • ! NOT logika pembalik. bernilai true bila kondisi false dan bernilai false bila kondisi true

Perulangan

digunakan untuk mengulang code dengan data yang banyak atau berbeda

  • For = digunakan untuk mengulang melalui blok kode beberapa kali.
  • While = digunakan untuk mengulang blok kode selama kondisi tertentu benar
  • Do While = mengeksekusi code satu kali sebelum memeriksa apakah kondisi benar, jika kondisinya benar maka akan mengulang loop.

Pengkondisian

digunakan untuk melakukan tindakan yang berbeda berdasarkan pada kondisi yang berbeda. Contohnya:

  1. if untuk menentukan blok kode yang akan dieksekusi jika kondisi yang ditentukan bernilai benar / true.
  2. else untuk menentukan blok kode yang akan di eksekusi jika kondisi yang di tentukan bernilai false / salah.
  3. else if untuk menentukan kondisi baru jika kondisi pertama salah.
  4. switch untuk melakukan tindakan yang berbeda berdasarkan pada kondisi yang berbeda (sama seperti elseif).

Fungsi

Fungsi javascript adalah blok kode yang dirancang untuk melakukan tugas tertentu.

aturan membuat fungsi:

  1. di definisikan dengan keyword function lalu di ikuti oleh nama_function dan tanda kurung ().
  2. nama fungsi dapat berisi huruf, angka, underscore (_), tanda dolar.
  3. tanda kurung termasuk nama parameter di pisahkan oleh koma (parameter1, parameter2, ....).
  4. kode yang akan dieksekusi ditempatkan di dalam tanda kurung kurawal {}.

Array

array merupakan variable khusus yang dapat menampung lebih dari satu data pada satu waktu.

syntax:

var buah = ["apel", "anggur", "jeruk", "pepaya"];

menghitung jumlah array

var nama_array = ["satu", "dua", "tiga", "empat"];
nama_array.length;
// outpunya = 3

mengakses data array

untuk mengakses eleme terakhir pada array, panggil menggunakan key elemen.

var buah = ['apel', 'anggur', 'jeruk'];
var buahSatu = buah[0];
// outputnya apel

mengubah array menjadi string (.toString)

nama_array.toString();

mengubah array menjadi string (.join)

apabila kita ingin mengganti pemisah dari defaultnya koma dengan yang lainnya, maka gunakan syntax berikut.

nama_array.join("pemisah_yang_di_inginkan"); bisa menggunakan spasi, koma, titik, tanda bintang, tanda plus, dan lain-lain.

mengeluarkan data dari array (.pop)

kita bisa mengeluarkan datat terakhir dari array dengan menggunakan syntax berikut.

nama_array.pop(); // mengeluarkan data dari array terakhir

memasukkan data array (.push)

nama_array.push("nama_item");

menggabungkan dua array (.concat)

var nama_array = array1.concat(array2);

menggabungkan tiga array (.concat)

var nama_array = array1.concat(array2, array3);

iterasi data dari array dengan (.forEach)

kita dapat looping menggunakan foreach.

iterasi menggunakan metode (.map)

kita bisa menggunakan metode map() untuk membuat array baru dengan melakukan fungsi pda setiap elemen array.

  • Metode map() tidak menjalankan fungsi untuk elemen array tanpa nilai.
  • Metode map() tidak mengubah array asli.

Object

merupakan abstraksi dari benda yang ada di dunia nyata. Digunakan untuk menyimpan data.

pada javascirpt, object terdiri dari nama dan value(variable) atau method (fungsi) atau bisa disebut juga dengan property.

analogi Object:

inisialisasi object dan property

var mobil = {nama: "toyota", type: "agya", berat: 600, warna: "putih"};

inisialisai method

mengakses property dan method pada object

nama_object.nama_method();
